Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • It is important to use the heading tags in order from one through six.
  • The src attribute defines the location, or source of the image

CSS

  • Cascading Style Sheets- To give the content style.
  • CSS describes how html elements are to be displayed on screen, paper, or in other media.
  • You can combine two selectors into a single CSS rule with a comma, allowing the declarations to affect both selectors.
  • A margin indicates how much space we want around the ooutside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Git is a version control system that lets you manage and keep track of each version of your project.
  • Commit changes means saving a new version of a file you're working on.
  • Markdown (.md) is the way content files are formatter in Github.
  • Pull request is asking for changes on a branch to merge into another branch.
  • Merge is when a PR is approved, changes are "Pulled in" or merged into the main branch.
  • Git status: checks what branch we are currently on.
  • Git checkout -b branch name: creates a new branch and switches to it
